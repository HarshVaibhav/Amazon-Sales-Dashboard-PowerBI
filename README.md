# Amazon-Sales-Dashboard-PowerBI
Interactive Amazon Sales Dashboard built using Power BI, DAX, and Power Query for sales, customer, product, and operational analysis.

## Project Overview

This project analyzes Amazon sales data using Power BI.

The dashboard provides insights into:

- Total Revenue
- Total Orders
- Total Customers
- Average Order Value
- Return Rate
- Product Performance
- Revenue by Category
- Revenue by State
- Payment Method Analysis
- Order Status Analysis

## Tools Used

- Power BI
- Power Query
- DAX
- Data Modeling
